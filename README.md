How it Works

Requirements
- Java (JDK or higher).
- IDE (e.g., VS Code, Eclipse, or IntelliJ).

How to Run
1) Copy the code into a Java file named "ElasticCollision.java".
2) Compile the program: "javac ElasticCollision.java".
3) Run the program: "java ElasticCollision".
